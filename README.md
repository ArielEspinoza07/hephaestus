# Hephaestus

> Forge powerful CLI tools with ease

🔨 A modern, type-safe wrapper around Symfony Console with exceptional Developer Experience.

---

## Philosophy

**Symfony Console is powerful but verbose.**

Hephaestus gives you:
- 🎯 Simple API for simple tasks
- 🔒 Type-safe everywhere (PHP 8.3+)
- ⚡ Fast to write, easy to test
- 🔧 Full Symfony power when needed
- 🏗️ Clean architecture with SRP
- 🧪 Test-friendly design

---

## Requirements

> - PHP 8.5+
> - Symfony Console 8.0+

---

## Installation

```bash
composer require arielespinoza07/hephaestus
composer dump-autoload
```
## Why "Hephaestus"?

Hephaestus (Ἥφαιστος) - Greek god of craftsmen, metalworking, and forging.

Just as Hephaestus forged legendary tools for the gods, this framework helps you forge powerful CLI tools.

🔨 **Forge. Build. Ship.**
